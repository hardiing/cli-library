# CLI Library App

A small Python application utilizing the command line to build a personal library database. Uses the ISBNLib library to facilitate verifying and looking up books by their
ISBN. The data then gets added to a JSON file. There is also a similar structure in place for a Wishlist functionality. You can also use the ISBN to perform a search for
a book to see if either the library or wishlist files contain that item.

## Getting Started

The following must be installed in your environment:

- python3
- isbnlib

1. Clone this repository

`git clone https://github.com/hardiing/cli-library`
`cd cli-library`

2. Execute build.sh to start program

`sh build.sh`
